Git e GitHub

O que é o Git?

O Git é um sistema de controle de versão distribuído, usado para gerenciar e acompanhar mudanças em arquivos de código.

O que é o GitHub?

O GitHub é uma plataforma online que hospeda repositórios Git e permite colaboração entre desenvolvedores.

Diferenças do Git e GitHub

O Git é a ferramenta de controle de versão, enquanto o GitHub é um serviço online que armazena e facilita o uso do Git na nuvem.

Principais comandos Linux para usar o Git Bash

  • ls → listar arquivos
  • cd → entrar em diretório
  • mkdir → criar diretório
  • rm → remover arquivo
  • pwd → mostrar caminho atual

Como criar um repositório no GitHub

No GitHub, clique em New Repository, dê um nome, escolha se será público ou privado e crie o repositório.

Como ligar um projeto existente na máquina

git init
git remote add origin URL_DO_REPOSITORIO

Como gerar uma versão com git

git add .
git commit -m "mensagem do commit"

Como enviar da máquina para o GitHub

git push -u origin main
